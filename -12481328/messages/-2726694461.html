與華夏合作是否需簽訂議定合約呢

哈嚕大家好！近期車機組開始與華夏合作，將 Eye Bus 的預約後台系統內嵌進現行的行控系統中，以方便調度員查看服務預約資訊時不需要頻繁地更換頁面，而是能夠直接經由行控系統中查看。
此合作預計花費 12 萬元（未稅），因金額超過十萬所以需要走採購形式，與前次和立皓合作時相同，皆需簽訂決標紀錄之合約，而與立皓合作時為確保未來可以順利擴張，我們有另訂「其他議定事項」要求立皓需提供與服務相關的部分原始碼，而針對此次與華夏的合作車機組認為是不需要另外簽署其他議定事項的，原因有以下幾點：
  1. 目前雙北的行控系統皆由華夏所設置並維運，因此若考量本案的施作範圍，是可以確保雙北營運的。（補充：目前宜蘭、雙北、桃園、台南、高雄之行控系統皆為華夏所建置）
  2. 各縣市之行控系統皆為客製處理，團隊現行所之預約系統介面是以雙北之行控系統來做設計，因此未來若有擴張至其他縣市之需求也未必能夠符合該縣市之行控介面。
  3. 目前偉大工程師志恒已完成預約系統的 API Document，並且此預約系統的主體皆為志恒的產出，華夏僅作銜接嵌入，若其他縣市需擴張使用，透過志恒所提供的 code 應為足夠。
根據以上三點，車機組這邊認為可以省略其他議定事項的簽訂。
如果各位有認為疏漏的地方煩請在 6/10 前告知～～我們可以再討論使否有簽訂其他議定事項的必要性！感謝大家！
