福利品

4. 我們在訪談過程中，發現使用者對於「福利品」的理解都不太一樣。

是否可以請 gary 協助我們了解以下問題：

a) 福利品營收佔所有ec營收的百分比？

  [gary] 福利品營收在總ec的百分比, by國家會不同. 如之前在EU的經驗, 福利品營收約佔六成, 因為當初EU賣的東西除福利品之外就是配件商品跟少量新機, 故佔比較高. TW的話, 在今年7/1日前, 福利品跟新品是分開兩個網站銷售, 且當時為了避免end user confuse有兩個華碩官方商店, 故MKT溝通導流主要以賣新機的網站為主(且當初新品的網站只有賣Zenfone與其週邊商品, 跟福利品筆電的價格有差異), TW福利品佔EC營收百分比約15-20%,僅供參考.

b) 因福利品認知差異、或誤購福利品（其實想買新品）的比例？

   [gary] 這個無從得知, 因為鑑賞期內end user無條件可以退貨, 他們也沒有告知理由(誤購, 看錯…等). 在EU我們有特別的頁面針對福利品說明, 教育end user什麼是福利品及其優點一直也是我們很想推的一個點.
