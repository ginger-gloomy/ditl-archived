智慧型手機普及，加上隨時隨地無線連網，行動應用與行動網站越來越多人使用。近期受到關注的焦點是 Google 調整了行動搜尋演算法，根據媒體報導 Fortune 500 大企業網站中的40% 將受到影響。如果您的Mobile Web 並未針對行動使用進行設計優化，未來將被 Google 行動搜尋調整排名往後。
因應行動化的趨勢與潮流，如何思考行動應用的產品與服務布局？
Mobile Web 與 Mobile App 之間該如何搭配？
使用者需求與情境有哪些差異？
如何評估該採用 RWD 響應式設計，還是獨立 Mobile Web？
從原本 Desktop Web 數據分析的經驗，又該如何轉移到 Mobile Web 流量分析？
該如何評估及優化 Mobile SEO？

上述問題將在 6/25-6/26「Mobile Web 實務研討：設計策略與數據分析」課程上，由六位專業講師提供實務經驗，供大家學習參考。
