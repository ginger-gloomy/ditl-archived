技術端分年進度與技術重點 總覽表

第一年

「實體介面壓覺」與「震動觸覺」

·      積木：堆積環境
·      積木加上震動  
·      String on Finger: 在手指產生觸控觸覺

 /「中央周邊視覺之圖文引導辨識」

·      文字系統、語言元素辨識力量測
·     符號、圖像的警告提示引導  
·      中央周邊不同位置明度、對比度之辨識力探討

--------------------------------------------------------------------------------

第二年

「力回饋本體感覺」

·      TRX 身體互動 – 各種運動類行 (划船, 滑水板, 拳擊)
·      或TRX綁在身上可無止盡往前跑

/「視覺化VR AR導覽技術 」

·     將ROI以子畫面拉到FOV內
·     視覺化方法空間引導的子畫面  

--------------------------------------------------------------------------------

第三年

「溫覺」互動 

·      由HMD面部溫覺
·      由HTC Vive 手把溫覺
·      由鞋墊或拖鞋式設備實現腳底溫覺

/「研發新周邊視覺技術與介面」

·      視覺技術與介面 for AR體驗
·      分析用戶移動時AR呈現資訊之閱讀效率
